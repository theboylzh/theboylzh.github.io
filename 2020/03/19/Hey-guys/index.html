Hey guys

这是我的第一篇文章

没什么，就测试一下。第一次自己用 Hexo + GitHub 的方式来搭建博客。值得纪念。在 Things 上面罗列了了解 GitHub 和搭建博客的 todo 之后，拖延了好多天。终于在某天，因为实在不想面对作品集设计——兴许是要动脑——我干脆捣鼓起了怎么搭建个人博客。说实在的，这是我从 17 年就想了解的事情了，如今 3 年过去了，百废待兴，我觉着要是不搞吧，就可太垃圾了。于是我马上撸起袖子，先从 Git 开始，再到 GitHub，再到 Hexo。其中使用印象笔记新版的编辑器可爽了。目前认为印象笔记中国版越来越牛逼，只需要等他把客户端也更新上新的编辑器，那可就太炸了。我目前个人最喜欢就是使用 Markdown + 表格的形式来总结知识，当前也只有印象笔记能满足我了。

当然，使用 Sublime Text 来写 Markdown 文档虽然没有 Bear 那么美观，但是感觉很硬核，自我良好的感觉爆棚，这也是好事情，会直接促进我继续深入研究那些“硬核”的事情。说实在的，由于 Bear 没有表格功能，我觉得很快可能就要弃坑了。谁叫印象笔记太好用了呢——至少它是可预期的，而 Bear 还不知道会否做这个功能。

最后，插个照片吧！
